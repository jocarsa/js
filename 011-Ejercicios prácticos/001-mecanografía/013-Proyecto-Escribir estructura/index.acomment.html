``` Este nuevo `div` actuará como un contenedor para mostrar el texto escrito por el usuario utilizando el teclado virtual. El contenido del `div` se actualizará dinámicamente mediante el archivo JavaScript `mecanografia.js`. El propósito de este elemento es mostrar al usuario el texto que va escribiendo en el teclado virtual, lo que proporciona una retroalimentación visual del texto que está siendo ingresado. El contenido de este `div` se actualizará en tiempo real a medida que el usuario presione las teclas en el teclado virtual. Esto permitirá al usuario ver lo que ha escrito hasta el momento y corregir cualquier error o realizar un seguimiento de su progreso mientras realiza ejercicios de mecanografía o cualquier otra actividad relacionada con la escritura.
